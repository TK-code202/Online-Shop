# _Online Market_

#### _This website was created for the purpose of prospective buyers ordering for desired goods._

#### By _**Tofunmi Kusimo**_

## Technologies Used

* _GIT_
* _HTML5_
* _CSS_
* _BOOTSTRAP 4.5.3_
* _Javascript and jQuery_


## Description

_This webpage collects input from buyers about information on goods being bought and generates a receipt._

## Setup/Installation Requirements

* _Clone this repository to your desktop_
* _Navigate to the top level of the directory_
* _Open the index.html file in your browser._
